# Quickstart Guide: Bedalo Village Website

## Project Setup

1. **Prerequisites**
   - Node.js (v18 or higher)
   - npm or yarn package manager

2. **Installation**
   ```bash
   # Clone the repository
   git clone https://github.com/inikknbedalo/bedalo.git
   cd bedalo

   # Install dependencies
   npm install
   ```

3. **Development Server**
   ```bash
   # Start the development server
   npm run dev

   # The site will be available at http://localhost:4321
   ```

## Content Management

### Using Decap CMS
1. Start the development server
2. Navigate to `http://localhost:4321/admin/`
3. Make content changes using the web interface
4. Changes will be saved to the appropriate content collections

### Content Structure
Content is organized in Astro content collections:
- `src/content/profil/` - Village profile and information
- `src/content/pariwisata/` - Tourism destinations
- `src/content/potensi/` - UMKM and village potential
- `src/content/kkn/` - KKN member information
- `src/content/struktur/` - Village leadership structure

## Content Collection Schema

All content collections are defined in `src/content/config.ts` with Zod schemas to ensure type safety and data validation.

### Adding New Content Types
1. Define a new Zod schema in `src/content/config.ts`
2. Create a new collection using `defineCollection()`
3. Add the collection to the `collections` export
4. Update the Decap CMS configuration in `public/admin/config.yml`

## Deployment

This is a static site generated by Astro, so it can be deployed to any static hosting service like Netlify, Vercel, GitHub Pages, or Cloudflare Pages.
